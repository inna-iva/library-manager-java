# Library Manager

This is a simple Java project for managing a library.  
It shows how books and users can be represented using classes and objects.  
The project uses basic OOP concepts like classes, objects, and Java Collections.

## Features

- Add books and users
- Track borrowed books
- Return books to the library
- Find books and users by ID
- Print all books and users

## Technologies

- Java
- OOP: classes, objects, getters/setters
- Collections: ArrayList
- Git & GitHub
- JUnit (planned for tests)
